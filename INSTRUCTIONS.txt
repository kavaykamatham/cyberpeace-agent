IF YOU GET ERRORS
═══════════════════════════════════════════════════════════════════════════════

ERROR: "Connection refused"
→ FIX: Make sure server is running in Terminal 1 (node server.js)

ERROR: "Invalid API Key"
→ FIX: Check .env file has correct GROQ_API_KEY and VIRUSTOTAL_API_KEY

ERROR: "Timeout"
→ FIX: Normal for first request. VirusTotal might be slow. Wait 5 seconds.

ERROR: Port 3000 already in use
→ FIX: Change PORT=3001 in .env file, restart server
