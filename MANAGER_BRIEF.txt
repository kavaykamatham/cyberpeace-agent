VIRUSTOTAL API INTEGRATION
═══════════════════════════════════════════════════════════════════════════════

How the System Works:

1. USER SUBMITS URL/FILE
   └─ Input goes to CyberPeace system

2. AI ANALYSIS (Groq)
   ├─ Analyzes text for scam indicators
   ├─ Detects phishing patterns
   ├─ Returns confidence score
   └─ ⏱️ Time: 1-3 seconds

3. VIRUSTOTAL SCAN (if suspicious)
   ├─ Submits URL/file to VirusTotal
   ├─ Gets analysis ID
   ├─ Waits for results
   └─ ⏱️ Time: 2-5 seconds (can take longer for new URLs)

4. RESULTS DISPLAY
   ├─ Shows detection from 70+ antivirus engines
   ├─ Displays malicious/suspicious/safe status
   ├─ Provides risk assessment
   └─ ⏱️ Status: "queued" (analyzing) or "completed" (ready)

WHY "QUEUED" STATUS:

When you see "Status: QUEUED" it means:
- ✅ URL was SUCCESSFULLY submitted to VirusTotal
- ⏳ VirusTotal is currently analyzing it
- ⏱️ Typically takes 2-5 seconds to complete
- 📊 Results will show up after analysis completes

This is NORMAL and EXPECTED for new URLs.

KEY FACTORS FOR SCAM DETECTION
═══════════════════════════════════════════════════════════════════════════════

The system checks multiple factors:

1. **LANGUAGE ANALYSIS**
   - Lottery claims ("You won...")
   - Urgency ("Click now!", "Limited time")
   - Suspicious offers
   - Poor grammar (common in scams)

2. **URL ANALYSIS**
   - Domain spoofing (fake-bank.com)
   - Suspicious TLDs
   - URL obfuscation
   - Bit shorteners (potential hiding)

3. **VIRUSTOTAL DATABASE**
   - 70+ antivirus engines
   - Known malware signatures
   - Phishing database
   - Real-time threat intelligence

4. **CONFIDENCE SCORING**
   - 0-100% confidence level
   - Based on multiple indicators
   - Conservative (avoids false negatives)
